# Linux Commands Cheat Sheet

## Navigation

```bash
pwd
```

Display current working directory.

```bash
cd ~
```

Go to home directory.

```bash
cd ..
```

Move to parent directory.

---

## File Management

```bash
mkdir folder
```

Create directory.

```bash
touch file.txt
```

Create empty file.

```bash
cp file1 file2
```

Copy file.

```bash
mv old.txt new.txt
```

Move or rename file.

```bash
rm file.txt
```

Delete file.

---

## Search

```bash
find . -name "*.txt"
```

Search txt files.

---

## Disk Usage

```bash
df -h
```

Filesystem usage.

```bash
du -sh folder
```

Folder size.
